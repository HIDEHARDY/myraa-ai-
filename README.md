# 👑 Suhani AI — Royal Neural Voice Companion

Suhani AI is a premium, real-time voice-to-voice neural companion application designed for low-latency, bidirectional audio communication. It operates exclusively using raw audio streams, enabling fluid, human-like conversations without the delays associated with text generation loops.

Styled with a rich, premium **Royal Purple, Electric Blue, Bright Gold, and Void Black** visual aesthetic, Suhani AI features an interactive avatar that responds dynamically to audio volume levels and user mouse movements.

---

## ✨ Features (What Suhani AI Can Do)

- **🎙️ Low-Latency Audio-to-Audio Connection**: Leverages a fast WebSocket connection for real-time, low-latency, bidirectional audio streaming.
- **🔊 Dynamic Interactive Equalizers**: Features a stunning tri-color responsive waveform equalizer that animates dynamically to speaker and companion volume.
- **👩‍🎨 Animated Avatar Companion**: Includes a vector avatar with 3D parallax head tracking following user cursor movement, eyes tracking, blinking, and lips syncing reactive to the voice volume.
- **🧠 Neural Memory Vault**: Saves and organizes continuous details, notes, tasks, and learned facts about the user across voice sessions.
- **⚙️ High-Tech HUD HUD Panels**: Live side panels featuring real-time stream status, latency tracking indicators, and context memory state metrics.
- **🛠️ Client Tool Integrations**: Autonomously trigger tool actions like opening specific web portals in new browser tabs, changing UI color palettes, rendering weather/fact holographic cards, or triggering celebratory confetti.

---

## ⚖️ Pros and Cons

### Pros
- **Natural Interaction**: Instant interruption handling and voice-to-voice communication creates a highly lifelike flow.
- **Breathtaking Design**: Custom glassmorphism layout with deep blacks, vibrant electric blues, royal purples, and gold shimmers.
- **Zero Text Pacing**: Communicates directly in voice chunks without waiting for text blocks to generate first.
- **Highly Responsive Controls**: Quick-toggle power indicator buttons, status indicators, and TLS secure link protocols.

### Cons
- **Internet Dependency**: Requires a stable high-speed network connection for seamless WebSocket audio rendering.
- **Hardware Access**: Relies heavily on microphone access permissions to start companion audio links.

## 🚀 How to Run Locally

### Prerequisites
- **Node.js** (v18+)

### Steps
1. Clone this repository and checkout the upgraded branch:
   ```bash
   git clone https://github.com/HIDEHARDY/myraa-ai-.git
   cd myraa-ai-
   ```
2. Install npm dependencies:
   ```bash
   npm install
   ```
3. Set your environment variable:
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
4. Start the application:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:5173`.
